# MoodMap

MoodMap is a mood-based nearby places recommender that suggests cafes, restaurants, and parks using real-time geolocation and open map data.

## Tech Stack
- React
- Leaflet
- OpenStreetMap
- Overpass API

## Features
- Live user location
- Mood-based place recommendations
- Interactive map with markers
- Distance calculation

## How to Run
```bash
npm install
npm run dev
